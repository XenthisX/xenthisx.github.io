Arduino Door Unlock System

GitHub

After moving into my dorm, and using a key for a week, I decided that there was a better way to unlock my door. Using an Adafruit NFC shield, a basic servo motor, lots of zip ties, and some programming, I was able to build an electronic door lock that isn't visible from the other side of the door, but can read MiFare cards through it.


The cardboard holding the servo over the lock is especially effective since it allows for minor flexibility when the servo actuates which takes some torque away from the lock holding mechanism, and the whole system is very simple, so the risk of failure is reduced. This lock now has internet access through a serial link to a laptop, and provides unlock and status check capabilities remotely.
