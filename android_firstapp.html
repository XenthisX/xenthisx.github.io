CREATING AN ANDROID APP ISN'T EASY

I decided to finally get into android development, as I’ve recently had more experience with larger scale application development, and I decided it would be good to document my journey.

    Currently I am experimenting with the Android TextClock as I want to make a clock that displays the time using text only. This will probably be a relatively complicated venture but I’m very excited to see how this sort of thing works on android. I already prefer the way UI development feels on Android as it seems to make a whole lot more sense than the previous experience I have doing UIs in Javafx.

    I started out following Google’s starting app development tutorial HERE and quickly created the application below. I also added the ability to tap on the clock and switch the mode from 12 hour to 24 hour, which is kind of a cool feature.

    Then I got sidetracked… I made the mistake of following a tutorial on how to make an ToolBar, which unknown to me has changed since the release of Android Marshmallow. I followed the tutorial and managed to make the app look like this:

Which is cool, but definitely not correct. At this point I decided the best thing to do might be to take out the old changes, so I began to learn how Marshmallow handles the “title bar”.


    And it was easy to fix! I just had to stop using the toolbar asset I just painstakingly created (It’s about 10 lines of XML) and switched the styles.xml file to use “Theme.AppCompat.Light.DarkActionBar" instead of "Themes.AppCompat.NoActionBar".

    I next decided to work on getting the menu for a help screen working, which I thought would be a good idea in order to become more familiar with launching tasks out of the main window. I also wanted to add a settings screen eventually, so knowing how to add items to the menu and then use them would be very useful.

    The help screen became far more complicated than I had initially anticipated as I wanted to add author information, but I managed to finalize it to have a couple of tabs. I then moved on to working on theming, which was exhausting as there is no real complete documentation about how to correctly theme Android apps.

    Creating a correctly theming program was actually so difficult for me I decided to create another blog about it that you can see HERE, it basically required me to learn a whole lot of Java and Android stuff I didn’t understand, while also working with old tutorials that weren’t exactly up to date.

That’s about where the development for this app finished, below are some of the final result pictures from the app, but I’ve decided to move on
